# Airbnb Listings & Reviews Analysis

## Project Overview

This project analyzes Airbnb listings and customer reviews across multiple global cities using Power BI. The dashboard provides insights into listing performance, pricing trends, customer ratings, and review activity to better understand market behavior and guest satisfaction.

## Tools Used

* Power BI
* DAX
* Data Visualization
* Excel

## Dashboard Pages

### Overview

Analyzes listing growth, room types, host activity, and city-level performance.

### Ratings

Explores customer satisfaction across cities using rating metrics such as cleanliness, communication, value, location, and overall rating.

### Reviews

Examines review distribution and customer engagement trends across cities.

## Key Insights

* Airbnb experienced rapid growth until 2015.
* Paris recorded the highest number of listings and reviews.
* Entire Place accommodations were the most popular room type.
* Hotel Rooms had the highest average price.
* Mexico City achieved the highest customer ratings.
* Cleanliness and value were important factors affecting customer satisfaction.

## Dataset

The dataset used in this project is the Airbnb Listings & Reviews dataset from Maven Analytics Data Playground.

Due to file size limitations, the original dataset is not included in this repository.
